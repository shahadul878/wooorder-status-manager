Create and manage custom order statuses for WooCommerce with workflow automation and customer tracking.

== Description ==

WooOrder Status Manager is a powerful WordPress plugin that extends WooCommerce with custom order statuses, workflow automation, and customer tracking features. Perfect for businesses that need more granular order management beyond the standard WooCommerce statuses.

= Key Features =

* **Custom Order Statuses**: Create unlimited custom statuses with colors, icons, and custom names
* **Workflow Automation**: Define automated workflows and rules for status transitions
* **Customer Tracking**: Let customers track their orders with a beautiful timeline view
* **Email Notifications**: Send automatic email notifications when status changes
* **Bulk Operations**: Update multiple orders at once with bulk actions
* **Admin Integration**: Seamlessly integrated with WooCommerce admin interface
* **Responsive Design**: Mobile-friendly tracking pages and admin interface
* **Shortcodes**: Use shortcodes to display tracking on any page
* **REST API Ready**: Built with extensibility in mind

= Use Cases =

* **Local Pickup Stores**: "Order Received → Packed → Ready for Pickup → Collected"
* **Food Delivery**: "Order Received → Preparing → Out for Delivery → Delivered"
* **B2B Wholesalers**: Track orders through multiple internal stages
* **Subscription Products**: Automate status changes on renewal/failed payments
* **Custom Workflows**: Any business with unique order fulfillment processes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wooorder-status-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure WooCommerce is installed and activated
4. Go to 'Order Statuses' in your WordPress admin menu to start creating custom statuses
5. Configure workflow rules and settings as needed

== Frequently Asked Questions ==

= Does this plugin work with WooCommerce Subscriptions? =

The free version provides basic compatibility. For advanced integration with WooCommerce Subscriptions, consider upgrading to the Pro version.

= Can customers see the custom statuses? =

Yes! You can control visibility for each status - admin only, customer only, or both. Customers can track their orders using the tracking page or shortcodes.

= How do I set up email notifications? =

Go to Order Statuses > Settings and enable email notifications. You can customize email templates for each status or use the default template with placeholders.

= Can I use this with other WooCommerce extensions? =

The plugin is designed to work with most WooCommerce extensions. For specific integrations with popular extensions, check the Pro version features.

= How do I create a workflow? =

Go to Order Statuses > Workflow Rules to create automated rules. You can set up triggers based on payment completion, stock updates, or manual actions.
